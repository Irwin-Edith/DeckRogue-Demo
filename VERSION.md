## 快速开始

### 场景入口

1. 打开 `Menu.unity` 作为初始场景
2. 主菜单提供以下入口：
   - **Play** → 进入角色选择
   - **Profile** → 进入存档管理
   - **Settings** → 进入设置界面

### 核心功能

- [X] 多场景导航系统
- [X] 画面设置（分辨率/帧率/全屏）
- [X] 音频设置（主音量/音乐/音效 + AudioMixer）
- [X] 三槽位存档系统
- [X] 退出确认弹窗
- [ ] 角色选择场景
- [ ] 设置数据持久化
